raspi-gps-tracker
=================

GPS tracker using Raspberry Pi

ingredients : 
- Raspberry Pi Model B (512 MB)
- GPS Module (EM 411)
- USB GSM Modem

Connection :

- Raspberry Pi -> ttyAMA0 (4800 baudra)-> EM-411
- Raspberry Pi -> ttyUSB0 -> USB GSM Modem

Software :
- GPSD
- FoxtrotGPS
- Gammu
- Script
